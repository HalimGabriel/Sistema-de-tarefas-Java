# Sistema de Tarefas em Java

Projeto CRUD de tarefas desenvolvido em Java utilizando:
- Programação Orientada a Objetos
- Enum
- List e ArrayList
- Scanner para interação no console

Funcionalidades:
- Adicionar tarefa
- Listar tarefas
- Concluir tarefa
- Remover tarefa

Projeto criado para prática de lógica e arquitetura básica em Java.
